# Code Style Rules - JobPilot

## NO EMOJIS IN CODE (NON-NEGOTIABLE)

### Forbidden files (ZERO emojis allowed)
- `.ts`, `.tsx`, `.js`, `.jsx`
- `.json`, `.yaml`, `.yml`
- `.sql`, `.env`
- `.css`, `.scss`

### Allowed files (emojis OK)
- `.md`, `.txt`

### Self-Check Before Every Write/Edit
Before writing to ANY file, verify:
1. Is the file extension in the forbidden list?
2. If YES -> scan for emojis and remove them
3. If NO -> proceed normally
